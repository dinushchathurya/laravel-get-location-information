# Laravel get user details based on the User's IP 

### In here it demonstrate how we can get details of the logged user using stevebauman/location package.

## Setup
 
```
$ git clone https://github.com/dinushchathurya/laravel-get-location-information.git
$ cd laravel-get-location-information
$ composer install
```

### Run The Service
```
$ php artisan serve
```

### Test Application

```
http://localhost:8000/location
```
